## 🌟 1. Tính năng chính
Ứng dụng **Vodka Web** cung cấp một giao diện hiện đại, trực quan và tối ưu cho trải nghiệm người dùng trong việc khám phá và thưởng thức điện ảnh:

### 🍿 Dành cho Người dùng (User Experience)
* 🏠 **Trang chủ (Home):** Hiển thị các bộ phim nổi bật, mẫu phim mới và danh sách xu hướng.
* 🔍 **Khám phá & Tìm kiếm:** Hệ thống tìm kiếm mạnh mẽ cùng bộ lọc linh hoạt theo thể loại, nhãn (tags) và đánh giá.
* 📖 **Chi tiết phim:** Xem thông tin đầy đủ về cốt truyện, diễn viên và nhận xét từ cộng đồng.
* ▶️ **Xem phim trực tuyến:** Trình phát video tích hợp hiện đại, hỗ trợ xem phim mượt mà.
* ⭐ **Hệ thống đánh giá:** Gửi bình luận, chấm điểm và tương tác trực tiếp với các người dùng khác.
* 👤 **Quản lý tài khoản (Profile):** Cập nhật profile, ảnh đại diện và theo dõi lịch sử hoạt động.
* 🔔 **Thông báo:** Hệ thống thông báo thời gian thực về các cập nhật và tương tác mới.
* 🔐 **Xác thực bảo mật:** Đăng nhập truyền thống, Google OAuth2 và các tính năng khôi phục mật khẩu bảo mật.

### 🛡️ Bảng điều khiển Quản trị (Admin Panel)
* 📊 **Tổng quan (Dashboard):** Thống kê trực quan về tăng trưởng phim, người dùng và lượt đánh giá.
* 🎬 **Quản lý nội dung:** CRUD phim chuyên sâu, quản lý cấu trúc mùa phim và tập phim.
* 📁 **Quản lý phân loại:** Quản trị hệ thống Thể loại (Genres) và Nhãn (Tags) linh hoạt.
* 👥 **Điều hướng người dùng:** Quản lý thành viên, phân quyền ADMIN/USER và trạng thái tài khoản.
* 💬 **Kiểm duyệt đánh giá:** Theo dõi và xử lý các nội dung bình luận từ cộng đồng.
* 📝 **Nhật ký hệ thống:** Theo dõi mọi thay đổi quan trọng thông qua Activity Logs.

## ⚙️ 2. Công nghệ sử dụng
Dự án được xây dựng trên stack công nghệ Frontend tiên tiến:
* **Framework:** Angular 21 (Latest)
* **Styling:** Tailwind CSS 4
* **State Management:** RxJS Streams
* **Auth:** Social Login & JWT Integration
* **Image Processing:** ngx-image-cropper
* **Build Tool:** Angular CLI

## 🚀 3. Cách cài đặt và chạy Project
Theo dõi các bước sau để khởi chạy **Vodka Web** trên máy của bạn:

### 📋 Yêu cầu hệ thống
* Đảm bảo máy đã cài đặt [Node.js](https://nodejs.org/) (Khuyên dùng phiên bản **v18+**).

### 💻 Các bước thực hiện

**Bước 1: Clone project về máy**
```bash
git clone https://github.com/chuonghoai/Vodka_Web.git
```

**Bước 2: Cài đặt dependencies**
```bash
cd Vodka_Web
npm install
```

**Bước 3: Khởi chạy Server**
```bash
npm start
# Hoặc
ng serve
```

**Bước 4: Trải nghiệm**
Mở trình duyệt tại [http://localhost:4200](http://localhost:4200). Ứng dụng sẽ tự động tải lại khi bạn thay đổi code.
